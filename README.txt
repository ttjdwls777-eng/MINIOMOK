FA Omok upgraded package

Files
- fa_omok_upgraded.js : upgraded full script

Main changes
1. Prevents the same player from joining their own room from another session.
2. Adds a dedicated room scene after creating or joining a room.
3. Locks menu movement while a room is active until Leave Room is pressed.
4. Keeps online play, friends, star wager, and leaderboard logic intact as much as possible.
5. Applies clearer scene switching between lobby / room / game.

How to use
- Replace your current script with fa_omok_upgraded.js
- If you want to keep a backup, save your old file separately first.
